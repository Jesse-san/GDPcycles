#The timeframe of the dataframe
start = datetime.datetime(1948, 1, 1)
end = datetime.datetime(2022, 10, 1)

#Attributing the names 
var_names=['GDPC1','PCECC96','GPDIC1','GCEC1','GDP','NETEXP','PCDG','DDURRG3Q086SBEA']
df_GDP = pdr.DataReader(var_names,'fred',start,end)

#Making durables and net exports as a nominal series and as a share of GDP respectivley  
df_GDP["Durables"] = df_GDP["PCDG"]/df_GDP["DDURRG3Q086SBEA"] 
df_GDP["Net exports"] = df_GDP["NETEXP"]/df_GDP['GDP'] 

var_names=['UNRATE'] 
df_urate = pdr.DataReader(var_names,'fred',start,end).resample('Q').first()/100

#Fixing indexes for unemployement 
index = df_GDP.index
df_urate.index = index

df_GDP["Unemployment"] = df_urate['UNRATE'] 

#Renaming the columns and replacing them with logical names 
#Removing unecessary columns  
df_GDP.rename(columns = {"GDPC1": "Real GDP", "PCECC96": "Consumption", "GPDIC1": "Investments","GCEC1": "Expenditures"}, inplace = True)
df_GDP.drop(columns = ['GDP', 'NETEXP','PCDG','DDURRG3Q086SBEA'],inplace = True)


#Creating 2nd dataframe excluding data after 2010
trim = 48
df_GDP2 = df_GDP[:-trim]

Prescott Filter¶

The Prescott filter was applied as a smoothening technique to remove short-term fluctuations that are commonly observed within the business cycle.

Net exports and unemployment were omitted from natural logs because of their association with negative numbers and percentages.

In [6]:
λ=1600

Y_cycle, Y_trend = sm.tsa.filters.hpfilter(np.log(df_GDP['Real GDP']), λ)
C_cycle, C_trend = sm.tsa.filters.hpfilter(np.log(df_GDP['Consumption']), λ)
I_cycle, I_trend = sm.tsa.filters.hpfilter(np.log(df_GDP['Investments']), λ)

def cor(x,y,xlag):
    m=len(x) 
    n=len(y)
    if not isinstance(xlag,int):  #check that we are using integers for lags or leads of x
        return('The lag length is not an integer.')
    if m != n: #check that the series match
        return('The series do not have the same length.')
    
    if xlag==0: #special case with no trimming
        x1 = x
        y1 = y
        # The next statement is the defintion of correlation
        return np.corrcoef(x1,y1)[0,1] #upper right element of the correlation matrix
   
    if xlag > 0:
        x1 = x[:-xlag] #trim off the end
        y1 = y[xlag:]  #trim off the beginning
        return np.corrcoef(x1,y1)[0,1]
    
    if xlag < 0:
        x1 = x[-xlag:] #trim off the beginning
        y1 = y[:xlag]  #trim off the end
        return np.corrcoef(x1,y1)[0,1]
    
    return('Nothing happened')

def color_negative_red(val):
    """
    Takes a scalar and returns a string with
    the css property `'color: red'` for negative
    strings, black otherwise.
    """
    color = 'red' if val < 0 else 'black'
    return 'color: %s' % color

def highlight_max(s):
    '''
    highlight the maximum in a Series yellow.
    '''
    is_max = s == s.max()
    return ['background-color: yellow' if v else '' for v in is_max]

Conclusions and Findings about the Business Cycle¶

Cycles in GDP are assumed to be the driver of economic activity. The effects of these cycles persist for about 4 quarters and is demonstrated through components of GDP. These findings can be visually explained by cross-correlation graphs which were implemented in this notebook.

Referencing the Post 2010 and Pre 2010 tables, the standard deviation on investment was 0.72 post 2010 and 0.75 pre 2010. Investment seems to be the most voltatile component of GDP which is followed closely by consumer durables with standard deviations of 0.048 post 2010 and 0.050 pre 2010. Unemployment appears to be a lagging indicator when compared to GDP since unemployment was low during economic booms, while net-exports are a leading indicator as a surge in imports was seen with economic booms.

Government spending demonstrates a positive correlation while having properties of a long-lagging indicator as it was high during a previous year where an economic boom was observed. Consumer durables demonstrated the properties of a leading indicator, since spending on durable goods were high when the economy "boomed" during the next quarter.
